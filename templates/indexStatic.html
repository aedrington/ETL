Jobs Versus Unemployment

Austin Edrington, Shade Petteway, Thihn Nguyen

Summary

We used a Monster.com Job Posting csv from Kaggle and pulled Unemployment and Population data via the Census API. This data was cleaned using Python, then imported into a MySQL database. We aggregated the Job data by State count, then compared to State wide unemployment data by merging the tables. We used MySQL because it is a relational database, and we were using States as a key to join the tables.
